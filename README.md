# Requirements

* Node.js, NPM
* MongoDB

# Launch

* Open [BotFather](https://t.me/BotFather) and register your bot
* Copy `.env.example` as `.env`
* Open `.env`, set `NODE_ENV` as `development`, put your bot token to `BOT_TOKEN`, put MongoDB connection options
* Run `npm install`
* Run `npm start`
